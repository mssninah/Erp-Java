Dans **Frappe HR**, le **Salary Structure** (structure salariale) joue un rôle crucial dans la gestion des salaires des employés. Il sert à définir les composants de rémunération (comme les gains et les déductions) d'un employé ou d'un groupe d'employés. Cela permet de calculer automatiquement les salaires lors de la génération des fiches de paie (**Salary Slip**). Voici une explication détaillée de sa logique :

---

### **1. Définition des composants**

Chaque structure salariale se compose de composants individuels, définis par :

* **Name (Nom)** : Le nom du composant (par ex., Salaire de base, Indemnité, Taxe sociale).
* **Abbr (Abréviation)** : Une abréviation pour identifier le composant rapidement.
* **Type** : Cela peut être :

  * **Earning** (gain) : Montants ajoutés au salaire (par ex., Salaire de base, primes, indemnités).
  * **Deduction** (déduction) : Montants soustraits du salaire (par ex., taxes, assurances, contributions sociales).
* **Valeur (Formula)** : Une valeur fixe ou une formule calculée basée sur d'autres composants.
* **Company** : L'entreprise à laquelle cette structure appartient.

---

### **2. Exemple : Gasy1**

Prenons l'exemple que tu as donné :

| Structure Salariale (gasy1) | Nom | Abréviation | Type               | Valeur     | Entreprise |
| --------------------------- | --- | ----------- | ------------------ | ---------- | ---------- |
| Salaire Base                | SB  | Earning     | `base`             | My Company |            |
| Indemnité                   | IND | Earning     | `SB * 0.3`         | My Company |            |
| Taxe sociale                | TS  | Deduction   | `(SB + IND) * 0.2` | My Company |            |

#### Logique :

1. **Salaire de base (SB)** : C'est le point de départ, une valeur fixe. Par exemple, 1000.
2. **Indemnité (IND)** : Calculée comme 30% du salaire de base :
   `IND = SB * 0.3`
   Si `SB = 1000`, alors `IND = 1000 * 0.3 = 300`.
3. **Taxe sociale (TS)** : Calculée sur la somme du salaire de base et de l'indemnité :
   `TS = (SB + IND) * 0.2`
   Si `SB = 1000` et `IND = 300`, alors :
   `TS = (1000 + 300) * 0.2 = 260`.

#### Résultat :

Le salaire net serait calculé comme suit :
**Net Salary** = `SB + IND - TS`
Dans cet exemple :
`Net Salary = 1000 + 300 - 260 = 1040`.

---

### **3. Fonctionnalité dans Frappe HR**

* **Standardisation** : Une **Salary Structure** permet de standardiser le calcul des salaires pour différents employés ou catégories (par ex., juniors, seniors, managers).
* **Flexibilité** : Les composants peuvent être dynamiques grâce aux formules.
* **Simplification** : Lorsque tu génères un **Salary Slip**, les montants sont automatiquement calculés selon la structure définie.
* **Transparence** : Les employés et les gestionnaires RH peuvent voir clairement comment le salaire est calculé.

---

### **4. Scénarios d'utilisation**

* **Gestion des primes et déductions** : Ajouter des primes (bonus de performance) ou des déductions (retards, absences, contributions sociales).
* **Conformité légale** : S'assurer que les salaires respectent les lois fiscales et les obligations sociales.
* **Budget et planification** : Estimer les coûts de main-d'œuvre en fonction des structures salariales.
